ConceptNet

An open, multilingual knowledge graph

ConceptNet is a freely-available semantic network, designed to help computers understand the meanings of words that people use. It originated from the crowdsourcing project Open Mind Common Sense, which was launched in 1999. It has since grown to include knowledge from other crowdsourced resources, expert-created resources, and games with a purpose.

ConceptNet in context

Linked Open Data

ConceptNet is a proud part of the ecosystem of Linked Open Data. You can use ExternalURL links in ConceptNet to find the same terms in other vocabularies, such as WordNet, DBPedia, and OpenCyc, which can provide you with other forms of information.

As a modern Linked Open Data resource, the data in ConceptNet is available in a JSON-LD API, a format that aims to make linked data easy to understand and easy to work with. If you don't care what JSON-LD is, it's just a JSON REST API with some extra metadata.

For information on how to use the ConceptNet API, see the API documentation. Or just start browsing it and you'll probably figure it out.

Word embeddings

ConceptNet is not just a set of "word embeddings", but it has been used to create word embeddings since before they were called that.

Word embeddings represent words of natural language as vectors that come out of some machine learning process. Recently, we've been using ConceptNet very effectively as a kind of universal glue for embeddings, helping to align word embeddings from different sources and in different languages to make a whole that is much better tahn the sum of their parts.

If you're excited about word embeddings, take a look at Conceptnet Numberbatch, the best word embeddings you can download as of 2016. These embeddings significantly outperform well-known embeddings, such as the word2vec embeddings (Google News skip-gram embeddings built using word2vec), on all evaluations we have tried. So far, they have continued to outperform newly published systems of word embeddings.
